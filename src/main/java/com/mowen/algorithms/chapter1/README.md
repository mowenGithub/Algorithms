算法:适合用计算机实现的解决问题的方法.  
数据结构:便于算法操作的组织数据的方法.是算法的副产物或结果.
 
    //最大公约数
    int gcd(int p, int q) {
        if(q == 0) return p;
        int r = p % q;
        return gcd(q, r);
    }
    
    //判定素数
    boolean isPrime(int N) {
        if(N < 2) return false;
        for(int i = 0; i*i <= N; i++) {
            if(N % i == 0) return false;
        }
        return true;
    }

#### 增长数量级的分类:

**|描述     |增长数量级  |说明     |举例   |**  
|常数级别    |1          |普通语句  |将两个数相加|    
|对数级别    |logN       |二分策略  |二分查找   |  
|线性级别    |N          |循环      |找出最大元素|  
|现性对数级别|NlogN      |分治      |归并排序    |  
|平方级别    |N^2        |双层循环  |检查所有元素对|  
|立方级别    |N^3        |三层循环  |检查所有三元组|  
|指数级别    |2^N        |穷举查找  |查找所有子集  |
